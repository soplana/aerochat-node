## nodeのビルドに必要なパッケージ

```
$ sudo aptitude install build-essential
$ sudo aptitude install libssl-dev
```

## node install

```
$ git clone git://github.com/creationix/nvm.git ~/.nvm
$ source ~/.nvm/nvm.sh
$ nvm install v0.9.5
```

## socket.ioをインストール

```
$ npm install socket.io
```

## run

```
$ node app.js
```
